# 🔎 FINDER

Professional Subdomain Recon Framework

Author: ZeKken

---

## 📌 Description

FINDER is an automated reconnaissance framework designed to discover and validate live subdomains efficiently.

It combines multiple powerful tools such as:

- Subfinder
- Assetfinder
- Sublist3r
- httprobe

The framework gathers subdomains from multiple sources, removes duplicates, and checks which ones are live.

---

## 🚀 Features

✔ Multi-source subdomain enumeration  
✔ Automatic dependency check  
✔ Live subdomain detection  
✔ Clean output  
✔ Automated installation  
✔ Professional terminal interface
