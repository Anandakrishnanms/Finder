## 📌 Description

FINDER is an automated reconnaissance framework designed to discover and validate live subdomains efficiently.

It combines multiple powerful tools such as:

- Subfinder
- Assetfinder
- Sublist3r
- httprobe

The framework gathers subdomains from multiple sources, removes duplicates, and checks which ones are live.

## 🚀 Features

✔ Multi-source subdomain enumeration  
✔ Automatic dependency check

## ## 💻 Installation

### 1. Clone the repository

```
git clone https://github.com/Anandakrishnanms/Finder.git
cd Finder
```

### 2. Run the installer

```
chmod +x install.sh
sudo ./install.sh
```

---

## ▶ Usage

### Run the tool

```
python3 finder.py
```

### Enter your target domain when prompted

```
Enter target domain: example.com
```

## 🔎 What Finder Does

  • Enumerate subdomains using multiple tools
